# Phoenix Services — Flight Path Intelligence

Phoenix Services is a high-performance, professional-grade aviation dashboard designed for weather-aware flight path optimization. It provides aviation professionals with real-time intelligence, enabling safer and more efficient route planning through advanced data visualization and AI-powered insights.

## 🚀 Key Features

- **Dynamic Flight Intelligence Map**: Integrated Leaflet-based mapping system with support for multiple weather and aeronautical layers.
- **Real-Time Weather Layers**: Toggle between various meteorological data overlays, including:
  - Precipitation (Radar)
  - Temperature Gradients
  - Cloud Cover
  - High-Altitude Wind Patterns
- **AI-Powered Route Optimization**: Compute the most efficient flight paths based on real-time atmospheric conditions to minimize fuel consumption and avoid turbulence.
- **Fuel Efficiency Analytics**: Detailed breakdowns of fuel burn projections and efficiency comparisons across different routes.
- **Interactive UI/UX**: A premium, NetJets-inspired design system featuring a sleek beige/cream palette, glassmorphism effects, and a responsive navigation rail.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla CSS3 (Custom Design System)
- **Mapping**: Leaflet.js
- **Typography**: Google Fonts (Inter & Outfit)
- **Icons**: Lucide Icons / Custom SVG
- **Data Integration**: OpenWeatherMap API & Aviation Weather APIs

## 📂 Project Structure

- `index.html`: The core application containing the UI structure, styles, and logic.
- `readme.md`: Project documentation and overview.

## 🏁 Getting Started

To run the dashboard locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/vrushalix/Pheonix_Airspace.git
   ```
2. Open `index.html` in any modern web browser.
3. Use the sidebar to navigate between Map, Route, Fuel, and AI Insight modules.
