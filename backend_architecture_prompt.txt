# MONDAY — BUSINESS OPERATIONS

## Role-Based Access Control (RBAC)

Create a complete RBAC architecture with the following roles:

### Public User

Can:

* Create account
* Login
* View personal dashboard
* Manage own profile
* Access platform features allowed by subscription

Cannot:

* View admin data
* Access employee tools
* Modify platform settings

### Employee

Can:

* View assigned users
* Access internal dashboards
* Manage customer support actions
* View operational metrics

Cannot:

* Access billing configuration
* Modify global system settings

### Admin

Can:

* Full system access
* User management
* Employee management
* Billing management
* Analytics access
* Platform configuration

### Required Deliverables

* Firebase custom claims implementation
* Middleware route protection
* Firestore security rules
* Permission checking utilities
* Protected API architecture
* Audit logs for admin actions
